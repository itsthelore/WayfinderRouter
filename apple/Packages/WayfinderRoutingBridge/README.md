# WayfinderRoutingBridge

Generated Swift bindings and a local XCFramework for Wayfinder's authoritative
Rust routing core.

The repository stores the package contract, handwritten marker, and Swift
tests. It does not commit generated Swift or binary artifacts. Build and verify
those products with:

```sh
apple/scripts/test_routing_bridge.sh
```

The build produces:

- `Artifacts/WayfinderRoutingFFI.xcframework` with Apple Silicon macOS, iOS
  device, and iOS Simulator slices;
- `Sources/WayfinderRoutingBridge/Generated/WayfinderRoutingBridge.swift`,
  generated by the pinned UniFFI toolchain.

The bridge is routing-only. Provider execution, credentials, persistence,
Apple Foundation Models, and UI stay outside this package.

`swift test` runs the generated binding against the macOS slice. The same
package is also compiled against the iOS device and simulator SDKs. A later
physical-device application slice must still prove runtime execution before
WF-ROADMAP-0016 Phase 1 exits.
